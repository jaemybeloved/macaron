# macaron

A Speech-To-Text Discord bot, powered by [OpenAI Whisper](https://openai.com/research/whisper).

## Prerequisites

- FFmpeg 4.4.2+
- Python 3.10.12+

```console
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
```
